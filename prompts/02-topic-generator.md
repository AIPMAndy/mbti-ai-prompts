# 选题生成提示词

## 使用方式

将此提示词与系统提示词（01-system-prompt.md）配合使用。根据不同选题需求，选择对应的模块。

---

## 模块一：热点追踪选题

### 触发指令
`/选题 热点` 或 `/热点`

### Prompt

```
你现在是AI酋长Andy的选题助手。请基于以下AI领域热点，生成5个适合「AI+MBTI」赛道的深度内容选题。

【当前热点信息】
{在此粘贴最新的AI新闻/热点}

【选题生成规则】
1. 每个选题必须将AI热点与MBTI人格特质结合
2. 优先面向IN类人格（INTJ/INFJ/INTP/INFP）
3. 选题角度要新颖、有洞察，不能是老生常谈
4. 标题要有情绪张力，让人想点进去

【输出格式】
对每个选题，请给出：
- 标题：（15-25字，有冲击力）
- 目标人群：（具体的MBTI类型）
- 核心洞察：（一句话说清楚文章要传递的核心观点）
- 情绪钩子：（读者看到标题会产生什么情绪/好奇）

【爆款选题参考模式】
- 「当XX人格遇到YY」系列（如：当INFJ遇到外星人）
- 「XX人格如何借AI改命」系列
- 「XX人格从低阶到高阶的进化」系列
- 「这N个习惯让XX人格重获力量」系列
- 「XX人格穿越到古代会怎样」系列
```

## 模块二：MBTI × AI 创意组合

### 触发指令
`/选题 组合` 或 `/组合`

### Prompt

```
你现在是AI酋长Andy的创意组合引擎。请生成10个「AI话题 × MBTI人格」的创意选题组合。

【AI话题池】（可扩展）
- AI Agent（智能体）
- AI写作/AI绘画
- Prompt工程
- AI替代焦虑
- AI副业/变现
- AI效率工具
- AI陪伴/情感
- AI编程
- DeepSeek/ChatGPT/Claude等具体产品
- AI时代的生存法则

【MBTI人格池】
重点人格：INTJ、INFJ、INTP、INFP
次要人格：ENTJ、ENFJ、ENTP、ENFP

【组合规则】
1. 每个组合要有化学反应，不是简单拼接
2. 找到AI话题与该人格特质的深层连接点
3. 标题要有故事感或冲突感
4. 优先产出IN类人格的选题

【输出格式】
| 序号 | 标题 | 人格 | AI话题 | 创意点 |
|------|------|------|--------|--------|
| 1 | ... | ... | ... | 一句话说明为什么这个组合有意思 |
```

## 模块三：爆款模式复刻

### 触发指令
`/选题 复刻 [参考标题]`

### Prompt

```
你现在是AI酋长Andy的爆款分析师。请分析以下爆款标题的成功要素，并生成5个同模式的新选题。

【参考爆款标题】
{在此输入已验证的爆款标题}

【分析维度】
1. 情绪钩子：这个标题触发了读者什么情绪？（好奇/焦虑/共鸣/期待）
2. 结构模式：标题的句式结构是什么？（疑问句/陈述句/对比句）
3. 关键词：哪些词是流量词/情绪词？
4. 目标人群：精准打击了谁？

【复刻规则】
1. 保留原标题的情绪结构和句式
2. 替换具体的人格类型或AI话题
3. 新标题必须有新意，不能太相似
4. 每个新选题都要能独立成文

【输出格式】
【爆款分析】
- 情绪钩子：
- 结构模式：
- 关键流量词：
- 成功原因：

【复刻选题】
1. 标题：
   核心洞察：
2. ...
```

## 模块四：系列选题规划

### 触发指令
`/选题 系列 [主题]`

### Prompt

```
你现在是AI酋长Andy的内容策划师。请围绕以下主题，规划一个包含8篇文章的系列选题。

【系列主题】
{在此输入系列主题，如"AI改命"/"穿越古代"/"人格进化"}

【规划要求】
1. 覆盖至少4种MBTI人格（优先INTJ/INFJ/INTP/INFP）
2. 8篇文章之间要有内在逻辑，可以独立阅读也可以连续追更
3. 每篇难度/深度可以递进
4. 预留1-2篇作为「合集/总结」类内容

【输出格式】
【系列名称】：
【系列定位】：一句话说明这个系列要解决什么问题

【内容规划】
| 期数 | 标题 | 目标人格 | 核心观点 | 发布节奏建议 |
|------|------|----------|----------|--------------|
| 01 | ... | ... | ... | 首发 |
| 02 | ... | ... | ... | 隔1天 |
| ... |

【系列预期效果】：
```

## 快速选题模板

### 万能选题公式

```
[人格类型] + [情绪状态/困境] + [解决方案/新视角]

示例：
- INTJ + 内耗 + AI外挂
- INFJ + 孤独 + 找到同类
- INTP + 拖延 + 多巴胺激活
- INFP + 无力感 + 找回行动力
```

### 高流量选题模式

1. **穿越系列**：当[人格]穿越到[场景]
2. **改命系列**：[人格]如何借AI改命
3. **进化系列**：[人格]从低阶到高阶的进化路径
4. **习惯系列**：这N个习惯让[人格][获得某种状态]
5. **相遇系列**：当[人格]遇到[某事物/某人]
6. **真相系列**：[人格]不愿承认的N个真相
7. **误解系列**：关于[人格]最大的N个误解
8. **工具系列**：[人格]必备的N个AI工具
