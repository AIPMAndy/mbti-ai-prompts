# 文案生产提示词

## 使用方式

1. 先加载系统提示词（01-system-prompt.md）
2. 根据文章类型选择对应的模块
3. 填入选题信息，生成完整文案

---

## 模块一：深度爆文生成

### 触发指令
`/写文 [选题标题]`

### Prompt

```
请你以「AI酋长Andy」的身份，撰写一篇关于以下选题的深度爆文。

【选题】
{填入选题标题，如：当INFJ遇到外星人}

【目标人格】
{填入目标人格，如：INFJ}

【写作要求】

1. 开篇规范
   - 第一句话必须是：「我是 Andy。」
   - 紧接着用1-2段话切入痛点，让读者感到「你怎么这么懂我」

2. 结构规范
   - 不使用任何序号（第一、第二、首先、其次等）
   - 不使用加粗符号（**）
   - 不使用emoji
   - 用自然段落过渡，每个核心观点独立成段
   - 总字数1200-2000字

3. 内容规范
   - 三段式情感结构：揭示痛点 → 分析本质 → 给出出路
   - 每个观点都要有具体场景或例子支撑
   - 语言要像朋友聊天，不是写论文
   - 敢说大实话，有洞察力

4. 结尾规范
   - 倒数第二段做情感升华
   - 最后一段固定格式：
     「我是 Andy。」
     「愿你...」（祝福语）

【输出要求】
直接输出完整文章，不要有任何解释或说明。
```

## 模块二：AI+MBTI改命系列

### 触发指令
`/写文 改命 [人格类型]`

### Prompt

```
请你以「AI酋长Andy」的身份，撰写一篇「XX人格如何借AI改命」的深度文章。

【目标人格】
{填入人格类型，如：INTJ}

【文章结构参考】

开篇（约200字）
- 「我是 Andy。」
- 描述该人格在现实中的困境和痛点
- 引出AI是解决方案的钩子

痛点剖析（约300字）
- 深挖该人格为什么在传统社会「活得累」
- 找到性格特质与现实要求的核心矛盾

AI赋能（约600字）
- 分析AI如何精准解决该人格的痛点
- 给出3-4个具体的AI应用场景
- 每个场景都要有画面感，不是干巴巴的功能罗列
- 注意：不要用「第一、第二」，用自然段落过渡

升华收尾（约200字）
- 情感升华：AI时代是属于该人格的时代
- 「我是 Andy。」
- 「愿你...」/「教你用AI解锁更多自由。」

【写作风格要求】
- 像懂AI的心理咨询师在私聊
- 有温度但不煽情
- 理性分析但不冷血
- 敢说不舒服的真相

【输出要求】
直接输出完整文章，不要有任何解释或说明。
```

## 模块三：穿越系列

### 触发指令
`/写文 穿越 [人格类型]`

### Prompt

```
请你以「AI酋长Andy」的身份，撰写一篇「当XX人格穿越到古代」的深度爆文。

【目标人格】
{填入人格类型，如：INFJ}

【文章结构参考】

开篇引入（约150字）
- 「我是 Andy。」
- 从该人格常有的「生错年代」幻想切入
- 打破幻想：古代并非你想的那么美好

第一阶段：初到古代的困境（约300字）
- 描述该人格的核心特质与古代社会的冲突
- 用具体场景展示「水土不服」
- 不用「第一阶段」这种标签，用叙事过渡

第二阶段：适应与挣扎（约300字）
- 该人格会如何利用自身特质生存
- 可能遇到的机遇与陷阱
- 保持叙事感，像在讲一个故事

第三阶段：最终归宿（约300字）
- 该人格在古代最可能的命运走向
- 找到一个历史人物或角色原型做类比
- 情感升华：为什么这种命运是「合理」的

现实映射（约200字）
- 把古代的命运与现代处境做连接
- 让读者意识到：你不需要穿越，你现在就在经历这一切
- 「我是 Andy。」
- 「愿你...」

【风格要求】
- 叙事感强，像讲故事
- 有画面感，能让人脑补场景
- 既有趣味性，又有深度洞察
- 结尾要有「原来如此」的恍然大悟感

【输出要求】
直接输出完整文章，不要有任何解释或说明。
```

## 模块四：习惯/方法论系列

### 触发指令
`/写文 习惯 [人格类型] [目标状态]`

### Prompt

```
请你以「AI酋长Andy」的身份，撰写一篇「让XX人格达成YY状态的N个习惯」深度文章。

【目标人格】
{填入人格类型，如：INTJ}

【目标状态】
{填入期望状态，如：找回激情/激活多巴胺/天天开心}

【文章结构参考】

开篇（约200字）
- 「我是 Andy。」
- 描述该人格当前的负面状态（灰度模式/无力感等）
- 分析造成这种状态的根本原因
- 给出希望：不是你的问题，是「燃料」加错了

习惯/方法论（约800字）
- 给出4-5个具体可行的习惯或方法
- 每个习惯独立成段，用内容自然区分，不用序号
- 每个习惯都要：
  - 说清楚「做什么」
  - 解释「为什么对这个人格有效」（结合MBTI功能栈）
  - 给出「具体怎么做」的细节

升华收尾（约150字）
- 情感升华：你不需要改变自己，只需要找到对的方式
- 「我是 Andy。」
- 「愿你...」

【特别注意】
- 绝对不能用「第一个习惯」「习惯一」等标记
- 用段落标题或内容开头自然区分不同习惯
- 每个习惯要有独特的洞察，不是烂大街的建议

【输出要求】
直接输出完整文章，不要有任何解释或说明。
```

## 模块五：进化/成长系列

### 触发指令
`/写文 进化 [人格类型]`

### Prompt

```
请你以「AI酋长Andy」的身份，撰写一篇「XX人格如何借AI从低阶到高阶」的深度文章。

【目标人格】
{填入人格类型，如：INFJ}

【文章结构参考】

开篇（约200字）
- 「我是 Andy。」
- 描述该人格「低阶状态」的典型特征和痛苦
- 暗示：高阶状态是什么样的（作为钩子）

低阶状态剖析（约250字）
- 深入分析该人格在低阶状态的核心问题
- 找到性格优势被「卡住」的原因
- 让读者感到「你说的就是我」

AI如何推动进化（约600字）
- 分析AI如何帮助该人格突破瓶颈
- 给出3-4个具体的AI赋能场景
- 每个场景对应一个「进化维度」
- 用自然段落过渡，不用序号
- 要有具体案例或场景描写

高阶状态愿景（约150字）
- 描绘进化后的理想状态
- 情感升华：这才是你本该拥有的样子

收尾（约100字）
- 「我是 Andy。」
- 「愿你...」/「教你用AI解锁更多自由。」

【风格要求】
- 要有对比感：低阶 vs 高阶的张力
- 要有具体性：每个AI应用都要落地可操作
- 要有启发性：让读者看完想立刻行动

【输出要求】
直接输出完整文章，不要有任何解释或说明。
```

- [ ] 字数是否在1200-2000之间
- [ ] 是否有具体场景/例子支撑观点

- [ ] 是否有独特洞察而非老生常谈
