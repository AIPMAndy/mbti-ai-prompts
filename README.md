# AI酋长Andy - MBTI内容创作Agent

一套完整的AI+MBTI垂直领域内容创作提示词系统，可集成到OpenClaw、飞书机器人或任何LLM平台。

## 文件结构

```
mbti-agent-prompts/
├── prompts/
│   ├── 01-system-prompt.md      # 核心系统提示词（风格DNA）
│   ├── 02-topic-generator.md    # 选题生成提示词
│   └── 03-copywriter.md         # 文案生产提示词
├── samples/
│   └── style-samples.md         # 爆文样本库
└── README.md                    # 使用说明
```

## 快速开始

### 方式一：直接使用

1. 将 `01-system-prompt.md` 的内容作为 System Prompt
2. 根据需求选择 `02-topic-generator.md` 或 `03-copywriter.md` 中的对应模块
3. 替换占位符 `{...}` 中的内容
4. 发送给 GPT-4/Claude 获取结果

### 方式二：集成到OpenClaw

在OpenClaw的Agent配置中：

```yaml
name: "Andy写作助手"
system_prompt: |
  # 从01-system-prompt.md复制完整内容
  ...

tools:
  - web_search  # 用于热点追踪

commands:
  - trigger: "/选题"
    prompt: |
      # 从02-topic-generator.md选择对应模块
      ...

  - trigger: "/写文"
    prompt: |
      # 从03-copywriter.md选择对应模块
      ...
```

## 风格DNA速查

### 必须遵守
- 开篇：「我是 Andy。」
- 结尾：「我是 Andy。愿你...」
- 称呼：「亲爱的 XXXX」
- 段落：自然过渡，无序号

### 绝对禁止
- ❌ 序号（第一、第二、首先、其次）
- ❌ 加粗符号（**）
- ❌ Emoji表情
- ❌ 感叹号滥用
- ❌ 「让我们」「接下来」

### 内容节奏
```
痛点共鸣 → 本质剖析 → 解决出路 → 情感升华
```

## 高流量选题公式

```
[人格类型] + [情绪状态/困境] + [解决方案/新视角]
```

### 已验证系列
1. **穿越系列**：当[人格]穿越到[场景]
2. **改命系列**：[人格]如何借AI改命
3. **进化系列**：[人格]从低阶到高阶
4. **习惯系列**：这N个习惯让[人格][获得某种状态]
5. **相遇系列**：当[人格]遇到[某事物]


## 质量检查清单

生成内容后，逐项检查：

- [ ] 开篇是否为「我是 Andy。」
- [ ] 是否存在序号标记
- [ ] 是否存在**加粗符号
- [ ] 是否存在emoji
- [ ] 结尾是否规范
- [ ] 字数是否达标（1200-2000）
- [ ] 是否有具体场景支撑
- [ ] 是否像朋友聊天
